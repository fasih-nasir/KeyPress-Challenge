## Starting the Game

How to Play

Start the Game:

Open the index.html file in your web browser.
Click on the "Start Game" button to begin.
Gameplay:

Random letters and numbers will appear on the screen.
Press the corresponding key on your keyboard as quickly as possible to score points.
If the screen width is below 992px (e.g., on mobile devices), the hidden input field will be automatically focused to allow input from the mobile keyboard.
Winning Condition:

Achieve a score of 50 or more within the 30-second time limit to win the game.
The game will display your progress and the time remaining during gameplay.
End of Game:

Once the time limit is reached, the game will display your final score and offer an option to play again.

Accessing with Keyboard Arrow Keys
The Traffic Race Game can be easily controlled using the keyboard arrow keys:

-Arrow Right Key: Move the car to the right.
-Arrow Left Key: Move the car to the left.
-Arrow Up Key: Move the car forward.
-Arrow Down Key: Move the car backward.

Mobile Screen Access
On mobile screens, arrow buttons will be displayed for easy control. Players can use these on-screen arrows to move the car left, right, up, and down, ensuring an intuitive and enjoyable gaming experience on any device.
